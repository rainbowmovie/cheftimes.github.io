== Description ==
Ultimate AMP is Magazine WordPress Theme. Ultimate AMP is a Child Theme of VideoStories.

Our Support: https://prowptheme.com/support/

== Installation ==

1. In your WordPress Admin Panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the ultimate-amp.zip file. Click Install Now.
3. Click Activate to use your new theme right away.
